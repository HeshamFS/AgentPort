# Roadmap

## Current

- Stabilize AgentPack `1.1.0`.
- Improve fixtures for Claude Code, Codex, and Gemini CLI.
- Collect real-world migration plans from early users.
- Keep local/private and managed scopes conservative.

## Next

- Add implemented adapters for Cursor and Windsurf.
- Expand research coverage for Antigravity, Kiro, Amp, OpenCode, Aider, Goose, Cline, Roo Code, Continue, Zed, JetBrains Junie, GitHub Copilot, and Amazon Q Developer.
- Add schema validation to CLI output.
- Add clearer migration reports for non-technical users.

## Later

- Support opt-in conversation summarization.
- Support adapter plugins outside the core repository.
- Publish compatibility reports as generated docs.
- Explore a small web UI for reviewing migration plans.
